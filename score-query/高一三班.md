# 成绩查询系统

## 项目说明
这是一个高一年级12月月考成绩查询系统，通过GitHub Pages部署。

## 功能特点
- 输入学生姓名即可查询成绩
- 显示总分和各科成绩
- 响应式设计，支持手机访问
- 无需后端支持，纯前端实现

## 使用说明
1. 访问：https://[您的用户名].github.io/score-query/
2. 输入学生姓名
3. 点击查询按钮

## 技术栈
- HTML5
- CSS3
- JavaScript (ES6)
- GitHub Pages

## 数据说明
数据来源：高一年级12月月考成绩表
包含科目：语文、数学、英语、政治、历史、地理
